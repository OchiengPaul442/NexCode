# Extensibility Structure

This folder provides a consistent layout for MCP-aware extensibility assets.

- skills: reusable capability descriptors and prompts
- tools: custom tool adapters and wrappers
- tasks: task blueprints and execution templates
- schemas: JSON schemas and shared payload contracts
- query: retrieval/query plans and ranking rules
- plugins: plugin manifests and runtime plugin hooks
- screens: UI metadata and structured view definitions
